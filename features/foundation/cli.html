CLI

Useful for generating a new project, and also provides generators for common files like components, actions and reducers.

Project Generation

This video demonstrates how to create a new project.

{% highlight bash %} lore new [project-name] {% endhighlight %}

Generators

The CLI can also generate project files such as components, actions and reducers. Files can be generated in ES5, ES6 or ESNext syntax (if you want to use decorators and static property types).
