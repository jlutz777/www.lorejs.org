generate component

CLI command to add a Component to your project.

Usage

{% highlight bash %} lore generate component [component-name] [options] {% endhighlight %}

Options

Option: ES Version

You can change which version of ES the component is generated in by passing in a language flag. By default, components are generated in ES5 syntax.

{% highlight bash %} lore generate component MyComponent --es5 lore generate component MyComponent --es6 lore generate component MyComponent --esnext {% endhighlight %} {% capture basic %}{% include features/usage/components/basic.md %}{% endcapture %} {{ basic | markdownify }}

Option: Router

Providing --router as an argument will generate a component configured for use with React-Router.

{% highlight bash %} lore generate component MyComponent --router {% endhighlight %} {% capture router %}{% include features/usage/components/router.md %}{% endcapture %} {{ router | markdownify }}

Option: Connect

Providing --connect as an argument will generatea a component wrapped on the lore.connect decorator.

{% highlight bash %} lore generate component MyComponent --connect {% endhighlight %} {% capture connect %}{% include features/usage/components/connect.md %}{% endcapture %} {{ connect | markdownify }}

Combining Options

Options can also be combined to generate a component containing each of those characteristics.

{% highlight bash %} lore generate component MyComponent --es6 --router --connect {% endhighlight %}
