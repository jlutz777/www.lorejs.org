Choose Your Own Adventure

The value proposition Lore provides varies a bit based on your comfort and experience building React applications. If you're just started out with React, or Web Development in general, you'll probably care more about how Lore makes it easier to learn React and build your first web application. If you've been developing for a while, and have worked on large single page applications before, you'll likely care more about what Lore includes to make it easier grow and support your application as it evolves over time.

To that end, we've grouped some helpful links below based on the experience level you identify with.

1. I'm new to React

Awesome! Welcome. You're about to embark on an amazing journey. To help give you some context about the value Lore provides, and what problem it's trying to solve for, you might be interested in these links:

  • What's it like to build a React app without Lore?
  • How does Lore make the experience easier?
  • Tutorial please! I'd like to try it out.

2. I've built one or more small React apps

Awesome! We should be friends. Since you already have some experience with React, you might enjoy these links to help you contrast your previous development experience with Lore.

  • Show me how Lore makes it easier.
  • Less talk, more code. Take me to the tutorial!
  • What does Lore provide to help me grow my application?

3. I've built or supported a large single page application

Awesome! We should grab coffee and swap stories, because truth be told, Lore is being built for you. These links should give you some idea of how serious Lore is about supporting large applications, and what it's doing to try and make your development experience more productive and fun.

  • Prove to me you understand that architecture matters.
