Lore Tutorial: Intermediate

Sadly, there is no tutorial yet for intermediate developers, which (for the sake of providing a definition) means you have a built a small React application before and have a general comfort with consuming data from API endpoints.

This page will instead convey what the tutorial will include once it exists. If you'd like to be part of the conversation and influence what will be in this tutorial, there there has been an issue created expressly for that purpose. Love to hear your thoughts!

Concepts to be Covered

This tutorial is expected to cover the following concepts (building on the concepts from the beginner tutorial):

  • Authentication
  • Launching dialogs
  • Extracting data from API responses wrapped in meta fields
  • Creating, Updating, and Deleting data
  • Pagination
  • Filtering
  • Displaying visual cues when data is being modified
