And that's because we haven't told the Layout where to render it yet. To fix this, open `Layout` and look in the render method for the code that looks like this: ```jsx
{/* Feed goes here */}
``` Modify that code to look like this: ```jsx
{React.cloneElement(this.props.children)}
``` When we added `Feed` to `routes.js` we listed it as a child of `Layout`. This code we just pasted says "clone my children and render them here".
