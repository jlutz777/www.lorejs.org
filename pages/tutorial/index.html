What You'll Build

This is the application we'll be building, the "Guessatron 5000". You type in the name of a color and submit it, and when you click on that color the route will change. The Guessatron will fetch that color, look at the name, and then attempt to show you a swatch of what that color actually looks like (it's all fake, don't get too excited).

Concepts Covered

This tutorial will cover the follow concepts:

  • Creating a new project
  • Creating React components
  • Connecting the application to an API server
  • Fetching the data components require
