Loading Screen

At the root of your project you should see an index.html file. This file is what gets served to the browser, and should currently look like this:

{% highlight html %} New Lore App

Loading...

{% endhighlight %}

The #root element is where your application is going to be attached. The default style tag and .loading-text element are there to serve as a minimalistic splash screen while the browser fetches the JavaScript files.
